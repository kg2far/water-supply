# 💧 Water Delivery Pro System

A full-stack water delivery web app (like Uber for water delivery 🚚💧) built with **Node.js, MongoDB, and Google Maps API**.

---

## 🚀 Features

### 👤 User Side
- Register & Login (JWT Authentication)
- Place water orders (liters, location, phone)
- Auto-detect GPS location 📍
- View personal orders
- Edit / Delete orders
- Navigate to delivery location (Google Maps)

### 👨‍💼 Admin Dashboard
- View all orders
- Update order status (Pending / Done)
- Delete orders
- Manage deliveries

### 🗺️ Map Features
- Live map view
- Order markers
- Driver route navigation
- Multi-order route optimization

### 📲 Notifications
- WhatsApp message trigger on order (basic integration)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Authentication:** JWT + bcrypt
- **Maps:** Google Maps JavaScript API
